# Access the Compliance API


The Compliance API is generally available to Enterprise plans, excluding Public Sector organizations.

Enterprise plan Primary Owners can enable the Compliance API by navigating to **[Organization settings \> Data and privacy](https://claude.ai/admin-settings/data-privacy-controls)** and clicking "Enable" under **Compliance API**.

Once the Compliance API is enabled, create new compliance access keys by clicking "+ Create key":


Creating a compliance access key will allow you to start pulling activity logs, chat data, and file content programmatically.

The Compliance API doc is accessible via **[our Trust Center](https://trust.anthropic.com/resources#692f35ca70113b25f1d2ec34:~:text=Request%20access-,Compliance%20API,-Anthropic%20Compliance%20API)**. Note that an NDA signature is required to access this.

## Audit log events in the Compliance API

The Compliance API now includes audit log events, giving you a full view across all your Claude deployments. To see which events are recorded via audit logs, refer to **[How to access audit logs](https://support.claude.com/en/articles/9970975-how-to-access-audit-logs#h_41cdad187a)**.
