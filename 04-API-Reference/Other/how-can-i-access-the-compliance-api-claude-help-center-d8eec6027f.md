# How can I access the Compliance API?

Updated yesterday

The Compliance API is generally available to Enterprise plans.

Enterprise plan Primary Owners can enable the Compliance API by navigating to [Admin settings \> Data and privacy](https://claude.ai/admin-settings/data-privacy-controls) and clicking "Enable" under **Compliance API**:

[](https://downloads.intercomcdn.com/i/o/lupk8zyo/1867698671/7de0faeaeae2a33d643970217689/619625b9-e2d9-4109-a76b-dfeb610be463?expires=1770585300&signature=b8f3afbd7941aec65722d014af224858c848b87081a8ae082537ee5a0f00d0e6&req=dSghEc93lYdYWPMW1HO4zbwCHRsmp8pPgysy9He2G7F3BuRuvg%2BwBbOJn0XO%0AAKVBgFTWItbumBep3lo%3D%0A)

Once the Compliance API is enabled, create new compliance access keys by clicking "+ Create key":

[](https://downloads.intercomcdn.com/i/o/lupk8zyo/1867701300/8a148e524f7ac2b1295d87005656/bd069690-3edf-4c51-ab66-09c73db81328?expires=1770585300&signature=a6d01c7e32c1657a55ae4e5caa2c3f37abe57b0c007818e8b2019a81619b1b29&req=dSghEc5%2BnIJfWfMW1HO4zdjcNEg50cgQLb9AVN3cSSHOJdBZgYmaRM7UNx8T%0Ayu7BwLXiiPP3MgwGAHg%3D%0A)

Creating a compliance access key will allow you to start pulling activity logs, chat data, and file content programmatically.

The Compliance API doc is accessible via [our Trust Center](https://trust.anthropic.com/resources#692f35ca70113b25f1d2ec34:~:text=Request%20access-,Compliance%20API,-Anthropic%20Compliance%20API). Note that an NDA signature is required to access this.

**Note:** The Compliance API is not available to Public Sector organizations.
