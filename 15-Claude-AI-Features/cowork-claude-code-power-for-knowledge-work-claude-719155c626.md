Find Cowork in your desktop app, alongside Chat and Code. Switch to Cowork when you want Claude to execute non-coding tasks.

### Share what you want to achieve

It all starts with your prompt and access to your tools and files. Open your desktop app, describe what you want help with, point Claude to your work, and it gets started on your task.

- **Desktop files:** Claude can access folders and files on your local machine
- **Connectors:** Pull in context from cloud tools like Google Drive, Notion, and more

## Stay in control

Cowork is in research preview. Agent safety is still in development. Take precautions while using it.

Read our safety guide

[Read our safety guide](https://support.claude.com/en/articles/13364135-using-cowork-safely)

Read our safety guide

### You decide what Claude can access

Cowork runs locally in an isolated virtual machine (VM) on your computer. You choose which folders and connectors Claude can see and it can't access anything without your explicit permission.

### Claude asks before acting

Always confirm before Claude handles financial, personal, or work-critical tasks.Before taking significant actions, Claude shows you what it plans to do and waits for your approval. You can redirect, refine, or take a different approach at any step.

### For Teams & Enterprise users

Cowork conversation history is stored locally on your device, not on Anthropic's servers. Enterprise features like audit logs, compliance API, data exports do not currently capture Cowork activity. Administrators can toggle it on or off at their discretion. Avoid granting access to local files with sensitive information, like financial documents.

## FAQ

### What’s the difference between Cowork and regular chat? 

In regular chat, Claude responds to your messages but can't access your files directly. In Cowork, Claude has permission to read, edit, and create files in folders you specify—so it can actually complete tasks rather than just describe how to do them.

### What's the difference between Cowork and Claude Code?

Desktop app users can start a task in [Claude Desktop](http://claude.com/download) and let it handle work in the browser without switching windows by enabling Claude in Chrome as a connector. Claude Code is a command-line tool designed for developers. Cowork brings the same agentic architecture to the desktop app, designed for non-coding knowledge work. No terminal required.

### What can Claude access? 

Only the folders and connectors you explicitly grant permission to. Cowork runs in an isolated VM, and you control network access through an allowlist in Settings.

### What kind of files can Cowork read, create, edit, and analyze?

Cowork can understand the following file types: 

- **Documents and text:** Word documents (.docx, .doc), PDF files (.pdf), Plain text (.txt), Markdown (.md), HTML (.html), JSON (.json), CSV (.csv), TSV (.tsv)
- **Spreadsheets:** Excel files (.xlsx, .xls, .xlsm)
- **Presentations:** PowerPoint files (.pptx, .ppt)
- **Images:** Common formats like .png, .jpg, .jpeg, .gif, .svg, .webp 
- **Data and config:** YAML (.yaml, .yml), XML (.xml), TOML (.toml)
- **Notebooks:** Jupyter notebooks (.ipynb)
- **Code files:** Pretty much any programming language: Python (.py), JavaScript (.js), TypeScript (.ts), React (.jsx, .tsx), Java, C/C++, Go, Rust, Ruby, PHP, SQL, shell scripts, etc.

You can also use Connectors to view additional files.

### What are the current limitations?

Cowork is available in the Claude desktop app for macOS and Windows and is a research preview. It doesn't yet support Projects, Memory across sessions, Artifacts, or Google integrations. Sessions don't sync across devices. **For Team and Enterprise plans: Cowork activity is NOT captured in Audit Logs, Compliance API, or Data Exports.**

### Does Cowork use more of my rate limit? 

Yes, agentic tasks consume more capacity than regular chat. Claude coordinates multiple sub-agents and tool calls to complete complex work.

### What are the current limitations for Team and Enterprise plans?

Cowork is available in the Claude desktop app for macOS and Windows and is a research preview. Cowork activity is not captured in Audit Logs, Compliance API, or Data Exports. Sessions don't sync across devices. It doesn't yet support Projects, Memory across sessions, Artifacts, or Google integrations.

### Can admins control Cowork access?

Yes. Admins can toggle Cowork off for their organization anytime in [Admin Settings](http://claude.ai/admin-settings/cowork).

### Is Cowork suitable for regulated workloads?

No. Do not enable Cowork for HIPAA, FedRAMP, or FSI regulated workloads.

### What are plugins?

Plugins are bundles of skills, connectors, slash commands, and sub-agents that customize Claude for specific roles and workflows. Instead of setting up each component individually, a plugin packages everything together — so Claude works like a specialist for sales, finance, legal, data, or any other function from the first conversation.
