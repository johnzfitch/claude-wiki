You can use PlanetScale to:

**Improve performance:**\
"What are the slowest queries in this database and suggest fixes"

**Perform queries:**\
"Get the latest 30 results from the ledger table"

**Track organizational information:**\
"Show me any currently outstanding invoices"

**Manage database structure:**\
"Show me the schema for the main branch and list all available branches"
