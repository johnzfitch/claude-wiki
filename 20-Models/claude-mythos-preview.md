# Claude Mythos Preview

Claude Mythos Preview is a new general-purpose language model announced April 7, 2026, that is strikingly capable at computer security tasks. It is not an incremental improvement but a step change in performance over Anthropic's current range of frontier models.

## Model Tier

Mythos Preview sits in a fourth model tier named **Copybara**, above the existing Opus, Sonnet, and Haiku tiers. Anthropic describes it as superior to any other existing AI frontier model.

## Benchmarks

| Benchmark | Score | Notes |
|-----------|-------|-------|
| SWE-bench Verified | 93.9% | Highest on record |
| GPQA Diamond | 94.6% | Highest on record |
| CyberGym | 83.1% | Highest on record |

## Cybersecurity Capabilities

During testing, Claude Mythos Preview demonstrated:

- **Zero-day discovery**: Capable of identifying and exploiting zero-day vulnerabilities in every major operating system and every major web browser
- **Vulnerability chaining**: Can chain together 3, 4, or sometimes 5 vulnerabilities in sequence to produce sophisticated exploits
- **Scale**: Has autonomously discovered thousands of zero-day vulnerabilities in critical software
- **Surpasses human experts**: Demonstrates a level of coding capability that can surpass all but the most skilled humans at finding and exploiting software vulnerabilities

## Availability

Claude Mythos Preview is **not publicly available**. Access is restricted to:

- **Project Glasswing partners**: AWS, Apple, Broadcom, Cisco, CrowdStrike, Google, JPMorganChase, Linux Foundation, Microsoft, NVIDIA, Palo Alto Networks
- **40+ additional organizations** that build or maintain critical software infrastructure
- Available through the Claude API, Amazon Bedrock, Google Cloud Vertex AI, and Microsoft Foundry for participants

### Pricing

Anthropic has stated the model is "very expensive for us to serve, and will be very expensive for our customers to use." Specific per-token pricing has not been publicly disclosed. Anthropic has committed $100M in usage credits for Project Glasswing participants.

## Context: Prior Cybersecurity Work

Mythos Preview builds on a trajectory of improving cybersecurity capabilities in Claude models:

- **November 2025**: Claude enters cybersecurity competitions, placing in top 25% of human competitors
- **January 2026**: Claude Sonnet 4.5 succeeds at multistage attacks on networks using only standard open-source tools (no custom toolkit needed)
- **February 2026**: Claude Opus 4.6 finds high-severity vulnerabilities at scale in open-source codebases
- **March 2026**: Claude Opus 4.6 discovers 22 Firefox vulnerabilities in two weeks (partnership with Mozilla)
- **April 2026**: Claude Mythos Preview announced with Project Glasswing
