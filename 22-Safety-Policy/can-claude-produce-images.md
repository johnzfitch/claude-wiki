# Can Claude produce images?


Claude doesn’t generate photos or illustrations the way image-generation tools do.

Claude can, however, build diagrams, charts, and interactive visuals directly in your conversation to assist with your understanding. These are generated using HTML and SVG—so you can interact with them, ask follow-up questions, and have Claude adjust them as you go. This feature is in beta on Claude web and desktop. For additional details, see **[Custom visuals in chat](../15-Claude-AI-Features/custom-visuals-in-chat.md)**.

Claude can also view and analyze images you upload. For more, see **[Uploading files to Claude](../15-Claude-AI-Features/uploading-files-to-claude.md)**.
