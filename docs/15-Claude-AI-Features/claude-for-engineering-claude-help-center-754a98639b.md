# Claude for Engineering

Updated yesterday

In this video guide, we'll explore several examples of how Claude can be used by engineering teams. Before you get started, review the availability of the features demonstrated in this video:

- Artifacts are available on all Claude plans.

- Projects are available to all users, including those with free Claude accounts. Free users can create a maximum of five projects.

- Project sharing features are available on Team and Enterprise plans. More information on project visibility and sharing can be found [here](https://support.claude.com/en/articles/9519189-project-visibility-and-sharing).
